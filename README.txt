This Project was created as the project assignment for the EL2320 - Applied Estimation.
Particle Filter was used in order to detect and track the eyes of a person,
Created by Timotheos Souroulla(timsou@kth.se) and Filip Hestell(hestell@kth.se).

To run the project follow these instructions:
1) Download the .zip files
2) Add the path with the images from the database to Matlab (\gi4e_database), including all the subfolders
3) Run the command 'EyeDetection_Gui' in the Command Window
4) Add all the parameters and select a person before running the algorithm
